# 🎵 Day by Music

> **"Five hundred twenty-five thousand six hundred minutes... How do you measure a year?"**

## 🕰️ 时间以什么为刻度？

是年龄的节点，是 100 天的纪念日，是人生不过三万天，还是回忆上次吃到 Chick-fil-A 薯格的日子已经过去了多久？

在音乐剧《吉屋出租》(RENT) 我最爱的歌曲 *Seasons of Love* 中，有句歌词写道：
> *How do you measure a year? In daynights, in sunsets, in midnights, in cups of coffee...*

**Day by Music** 诞生于一个浪漫的设想：如果我们以**音乐 (Songs)** 来衡量时间呢？

---

## 🌟 项目简介

**Day by Music** 是一个轻量级的网站，致力于通过音乐重塑我们对时间的感知。它会展示历史上“那年今日”有哪些歌曲或专辑发布了。

它不仅是一个技术工具，更是一份感性的礼物。让每一个平凡的日子，都能通过一段跨越时空的旋律，找到属于它的独特注脚。

## ✨ 核心功能

* **📅 那年今日 (On This Day)：** 自动匹配当前日期，回溯历史上同月同日发布的经典音乐作品。
* **🎶 旋律刻度：** 不同于冷冰冰的数字，用音乐人创作的瞬间来标记你的生活节点。
* **🎨 极简设计：** 界面干净纯粹，让视觉焦点回归到唱片封面与旋律本身。

## 🛠️ 技术栈

* **Framework:** React / Next.js (或相应前端框架)
* **Deployment:** [Replit](https://replit.com/)
* **Data Source:** Music Metadata APIs
* **Styling:** Tailwind CSS / CSS Modules
